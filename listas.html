Listas

Funcionamiento

Las listas son usadas para resumir informacion y hacerla mas clara.
Las listas mejoran en SEO.
Existen del tipo ordenadas, no ordenadas y descriptivas.

Listas Ordenadas

Emplean la etiqueta <ol> ordered list (lista ordenada) y la sub etiqueta <li> que significa list item (elemento de lista)

Ejemplo; una lista de paso a realiza:

  1. Sacar la llanta de refaccion
  2. Sacar la llave de cruz
  3. Aflojar los birlos
  4. Quiar llanta ponchada
  5. Poner llanta refaccion

Listas No Ordenadas

Emplean la etiqueta <ul> unordered list (lista no ordenada) y la sub etiqueta <li> que significa list item (elemento de lista)

Ejemplo; una lista de cosas sin orden especifico:

  • Peras
  • Mangos
  • Plantano
  • Manzana
  • Pi침aa
