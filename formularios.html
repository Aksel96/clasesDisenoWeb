Formularios

Introduccion

Es un control creado con el tag FORM, el cual permite obtener informacion de entrada desde el usuario


Atributos importantes
  • action: Establece la URL destino de los datos
  • METHOD: Establece el verbo del protocolo HTTP con el cual se enviara la informacion. Los mas usados son: GET y POST
  • name: Asigna un nombre al formulario para ser recuperado del lado del servidor
